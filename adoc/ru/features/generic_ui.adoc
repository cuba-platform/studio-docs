[[generic_ui]]
=== Универсальный пользовательский интерфейс

Studio предлагает широкий набор возможностей для оптимизации работы с {main_man_url}/gui_framework.html[пользовательским интерфейсом] приложения.

В секции *Generic UI* дерева проекта вы можете найти следующие элементы:

* <<generic_ui_menu,Web Menu>> открывает графический редактор главного меню приложения.

* {main_man_url}/main_message_pack.html[Main Message Pack] содержит строки сообщений для элементов главного меню и общих элементов UI приложения.

* <<screen_descriptor,Screens>> отображает созданные экраны.

* <<generic_ui_themes,Themes>> используется для управления темами визуального представления приложения.
